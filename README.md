总体架构
======
* 域名 book.54php.cn
* 管理后台地址  book.54php.cn/web
* 会员端地址 book.54php.cn/m


文件架构解释
======
* controllers  
* models  数据表
* modules 模块 分为 web , m ,weixin
* views  视图
* web 静态文件

web目录详细解释
=========

* web/css/web  web端css文件
* web/css/m  m端css文件
* web/font-awesome  awesome字体库
* web/fonts  业务使用字体
* web/images/web
* web/images/m
* web/js/web
* web/js/m
* web/uploads 上传文件目录
